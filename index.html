Cruise Ship

I made an interactive cruise-ship where the user can create their own itinerary.
I learnt about Test Driven Development and DOM manipulation - I created different types of objects that could interact with eachother.
This was the first time I had to get user input data from a form - it was super challenging at first.

Virtual Pet

This was my first javascript project!
Using test driven development, I created dynamic javascript objects with properties such as age, hunger and fitness. Next, I designed and created a user interface with a choice of 3 different pets and unique animations for each one.
To play this game, you need to feed and play with the pet to keep it alive!

Weather App

This was my first project built with react!
This is an app that can pull weather forecast data from an API. I also added a search function that allows the user to search for the weather anywhere in the UK.
I then designed and created a user interface that (i think) looks really pretty.

Music Library

This is a music library API built with Node.js, Express.js and PostgreSQL.
This API was built using test-driven development, and the testing was done with Mocha, Chai and Supertest.
This was made during the back-end module at Command Shift Bootcamp, and working on this project has made me feel much more comfortable working with APIs.
